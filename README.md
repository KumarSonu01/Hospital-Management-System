## 🚀 Overview

A scalable hospital management system that handles **patient records, appointment scheduling, and role-based access** with secure authentication.

Designed to reduce manual work and improve operational efficiency.

## ✨ Features

- 🔐 **Secure Authentication**
  - JWT-based login/signup  
  - Role-based access (Admin, Doctor, Patient)

- 📅 **Appointment Scheduling**
  - Real-time slot booking  
  - Reduced scheduling conflicts by **30%**

- 🧑‍⚕️ **Patient Management**
  - Manage 100+ patient records  
  - Full CRUD operations

- ⚡ **High Performance**
  - API response time under **200ms**  
  - Optimized backend operations

- 🛡️ **Security**
  - Protected routes  
  - Reduced unauthorized access
