🏥 Hospital Management System

A full-stack web application designed to streamline hospital operations including patient management, appointment scheduling, and role-based access control.

🚀 Overview

This system digitizes core hospital workflows to reduce manual effort, improve efficiency, and ensure secure handling of patient data.

Built using the MERN stack, it supports multiple user roles and real-time appointment booking.

⚙️ Tech Stack
Frontend: React.js
Backend: Node.js, Express.js
Database: MongoDB
Authentication: JWT (JSON Web Token)
API: RESTful APIs
✨ Key Features
🔐 Authentication & Authorization
Secure login/signup using JWT
Role-based access (Admin, Doctor, Patient)
📅 Appointment Management
Real-time slot booking
Conflict reduction in scheduling (~30%)
🧑‍⚕️ Patient Management
Handles 100+ patient records efficiently
CRUD operations for patient data
⚡ Performance Optimized
API response time under 200ms
Smooth UI with fast interactions
🛡️ Security
Protected routes
Reduced unauthorized access risks
📊 Impact
Reduced manual scheduling conflicts by 30%
Improved workflow efficiency for hospital staff
Secure and scalable architecture for real-world usage
